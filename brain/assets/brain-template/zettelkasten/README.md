# Zettelkasten

Slip-box for atomic, linked notes. Organize by linking, not by folders.

- **[zettelkasten.md](zettelkasten.md)** — The Zettelkasten method (principles, linking, structure, writing). For both humans and agents.
- **inbox/** — Quick captures and rough ideas. Process into permanent notes in this folder.
- **template-zettel.md** — Copy this when creating a new note. Use tags and markdown-style links `[text](path/to/note.md)` (no wiki links).
- **AGENTS.md** — Agent-focused scope and repo conventions; points to zettelkasten.md for the full method.

Permanent notes live directly in `zettelkasten/`. Name files **date+slug** (e.g. `2025-03-13-jupiterone-graph-queries.md`): `YYYY-MM-DD` then a kebab-case slug.
