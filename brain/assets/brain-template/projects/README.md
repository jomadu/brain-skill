# Projects

Time-bound work with a clear outcome. One folder per project.

- **Agent guidance:** See [AGENTS.md](AGENTS.md) for scope and conventions under this folder.
- **Active:** Add project folders here (e.g. `my-project/` with a `README.md` and any subnotes).
- **Done:** Move completed projects to `archive/<project-name>/`.

Use standard markdown links to connect to [areas](../areas/) (e.g. relationships), [resources](../resources/), and [zettelkasten](../zettelkasten/) notes.
